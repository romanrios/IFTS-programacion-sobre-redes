# Clase 7 - Ejercicio Cliente-Servidor

Ríos, Román | Comisión 3A

## Descripción

Aplicación cliente-servidor para la gestión de notas utilizando sockets y formato JSON para la comunicación.

El cliente permite ver, agregar y eliminar notas, mientras que el servidor procesa las solicitudes y gestiona los datos.

En este ejercicio se proponen mejoras sobre un código base.

## Mejoras implementadas
- Soporte multicliente con threading
- Persistencia de datos con SQLite
- Manejo de errores en cliente y servidor
- Validación de datos de entrada

## Ejecución

Servidor:
```bash
python servidor_mejora.py
```

Cliente:
```bash
python cliente_mejora.py
```
