# Clase 4 - Ejercicio Cliente-Servidor

Ríos, Román | Comisión 3A

## Descripción

Ejercicio orientado a mejorar un código base, analizando qué agregar o modificar para lograr una mejor interacción y respuesta en un modelo cliente-servidor.


## Mejoras implementadas
- Interacción continua (múltiples preguntas)
- Soporte multicliente con threading
- Comando de salida ("salir")
- Normalización de texto
- Manejo básico de errores
- Mensaje de bienvenida

## Ejecución

Servidor:
```bash
python servidor_mejora.py
```

Cliente:
```bash
python cliente_mejora.py
```
